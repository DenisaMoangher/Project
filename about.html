E-library

Biblioteca virtuala este o bibliotecă mobila, nu trebuie să meargi fizic la biblioteca poti avea acces la informatii oricand si oriunde. Într-un mod placut, atat timp cat este disponibila o conexiune la Internet poti avea acces 24/24h. În acelaşi timp, aceleasi resurse pot fi utilizate simultan de o serie de cititori pur şi simplu iar utilizatorul poate folosi orice termen de cautare.

E-library

E-library este o aplicatie web construita pentru gestionarea unei biblioteci online. Aceasta urmareste gestionarea atat pentru administrator cat si pentru utilizator. Prin aceasta aplicatie se doreste o gestionare intr-un mod cat mai usor si eficient pentru procesul de imprumutare al cartiilor
